# Compute Sponsoring

Compute Sponsoring defines a new category for advertising in generative AI.
It proposes a new reference point for advertising attribution, together with clear principles and a notation system for transparent declaration.
[Compute Sponsoring v1.0](https://github.com/compute-sponsoring-org/Compute-Sponsoring/blob/main/whitepaper/Compute%20Sponsoring%20v1.0.pdf)

## Status

Compute Sponsoring is currently at **Version 1.0** and published as an open standard proposal.  
Version 1.0 defines the category, core principles, and classification framework.  

## Feedback, Discussion and Input

You are welcome to discuss, critique and help further develop Compute Sponsoring in the following discussion thread:
[Concept and Evolution of Compute Sponsoring](https://github.com/compute-sponsoring-org/Compute-Sponsoring/discussions/2)

## Versioning

The document follows a simple versioning structure:

- **v1.0.x** — editorial clarifications, wording improvements, non-structural changes  
- **v1.x** — extended clarification, additions, or refined definitions (no structural break)  
- **v2.0** — structural revisions, formalization, or technical specification  

## Governance (in preparation)

The formal governance model is currently under development. Proposals and suggestions are explicitly welcome.

Discussion takes place in GitHub Discussions under the thread:
[Governance & Process](https://github.com/compute-sponsoring-org/Compute-Sponsoring/discussions/1)

The goal is to establish a governance model that ensures:
- transparency   
- integrity of revisions  
- independence of category development  

## Conflicts of Interest

Compute Sponsoring is an open proposal. Contributions are welcome regardless of institutional or commercial affiliation.
